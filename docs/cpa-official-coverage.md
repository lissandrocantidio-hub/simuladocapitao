## Critérios de importação

- Cada questão precisa funcionar sozinha dentro do simulado.
- Quando a prova usa contexto compartilhado, o enunciado deve ser reescrito para incorporar os dados necessários.
- Questões que dependem de figura, carta, rosa de manobra ou imagem específica devem receber anexo local ou reescrita suficiente antes de entrar no banco.
